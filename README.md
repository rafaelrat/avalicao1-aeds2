Avaliação 1 da disciplina de Algoritmos e Estruturas de Dados II

## Requisitos de Ambiente

O projeto foi desenvolvido em Python. Para executar os experimentos e gerar os gráficos, é necessário instalar as dependências de manipulação de dados e plotagem gráfica.

```bash
# Crie e ative um ambiente virtual (opcional, mas recomendado)
python -m venv venv
source venv/bin/activate  # Linux/macOS
# venv\Scripts\activate   # Windows

# Instale as dependências numéricas e gráficas
pip install -r requirements.txt
```

## Execução

Toda a execução é feita pelo `notebook.ipynb`
